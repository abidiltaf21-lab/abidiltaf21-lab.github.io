Smooothpixel documentation
============================

For full project overview:
  - English: README.md (repository root)
  - Pashto:  PROJECT_OVERVIEW.md (this folder)

To run the site locally:
  1. Start API:  dotnet run --project Project/ReactApi
  2. Start UI:   cd smooothpixel/souce && npm install && npm run dev
  3. Open browser at the URL shown by Vite (usually http://localhost:5173)

Legacy note: open index.html only applies to static builds; use npm run dev for development.
